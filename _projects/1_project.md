Cloud processing can significantly change aerosol particle properties. During my PhD, I used the particle-resolved model PartMC-MOSAIC to investigate how cloud processing affects aerosol mixing state and optical properties.

## Key Findings

- Cloud processing homogenizes aerosol mixing state
- Resuspended particles after cloud evaporation have different optical properties
- These changes have implications for aerosol radiative forcing estimates
